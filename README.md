### COMMAND LINE SUPERMARKET STORE

### ABOUT
    A command line supermarket store that allows, purchases, adding items to cart and store,
    viewing items in market and cart.

### TECHNOLOGIES USED:

    * C++

### FEATURES

    * View Stock     
    * Add To Stock  
    * Purchase items
    * View and Edit Cart
    * Search Items
    * Clear Cart
